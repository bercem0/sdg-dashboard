# 🌍 SDG Dashboard

Een interactief data-dashboard dat inzicht geeft in wereldwijde duurzaamheidsdoelen (Sustainable Development Goals). Dit project visualiseert belangrijke statistieken en trends, zodat gebruikers complexe data sneller kunnen begrijpen en analyseren.

## 🚀 Over het project

Het SDG Dashboard is ontwikkeld met **Next.js** en richt zich op het bouwen van een moderne, schaalbare en gebruiksvriendelijke webapplicatie. Door middel van datavisualisaties kunnen gebruikers eenvoudig informatie verkennen over verschillende mondiale thema’s zoals gezondheid, onderwijs, energie en stedelijke ontwikkeling.

De applicatie is ontworpen met aandacht voor performance, toegankelijkheid en duidelijke UI/UX.

## 🎯 Actieve Sustainable Development Goals

Dit dashboard focust op de volgende SDG’s:

* **SDG 3 – Goede gezondheid en welzijn**
  Visualisatie van gezondheidsstatistieken en indicatoren die bijdragen aan een beter wereldwijd welzijn.

* **SDG 4 – Kwaliteitsonderwijs**
  Data over toegang tot onderwijs en wereldwijde leertendensen.

* **SDG 7 – Betaalbare en duurzame energie**
  Inzichten in hernieuwbare energie en duurzame energiebronnen.

* **SDG 11 – Duurzame steden en gemeenschappen**
  Analyse van stedelijke groei, leefbaarheid en duurzame ontwikkeling.

Het doel van deze applicatie is om complexe datasets om te zetten in duidelijke visuals, zodat beleidsmakers, studenten en andere geïnteresseerden sneller inzichten kunnen verkrijgen in de voortgang van duurzame ontwikkelingsdoelen.

## 🔧 Installatie

Clone de repository:

```bash
git clone https://github.com/bercem0/sdg-dashboard.git
```

Ga naar de projectmap:

```bash
cd sdg-dashboard
```

Installeer dependencies:

```bash
npm install
```

Start de development server:

```bash
npm run dev
```

Open daarna **[http://localhost:3000](http://localhost:3000)** in je browser.
